You can see the source code for this website at http://github.com/groovymarty/gvyweb.

I wrote all the code myself. It wasn't easy, and sometimes I had to think really hard.

The backend server is running on a Digital Ocean droplet. Pictures, video and metadata files are stored on DropBox. The backend fetches them from DropBox and serves them as requested. DropBox also provides image compression, which is very nice of them because their giant servers can do it really fast.

The source for my backend server is at http://github.com/groovymarty/gvypics.
